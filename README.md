# FASN-1
Loss of ptr-6 restores eggshell integrity and embryonic viability in C. elegans fatty acid synthase mutants


Fatty acid synthase (FASN) is a key rate-limited, dimeric multi-enzyme complex in the de novo lipogenesis pathway. 
We performed an EMS mutagenesis screen in the temperature-sensitive fasn-1(g43ts) mutant to identify mutations that suppress embryonic lethality.
Each independent suppressor strain was sequenced by WGS to identify the causal mutation.
Variant call in vcf format is provided.
